Features

MATIVATED

is a set of tools for Individuals and Clubs willing to improve in Jiu Jitsu.

Training Sessions

Save training type, length, techniques etc. in BJJournal. Measure your progress and check statistics.

Clubs

Join Club with your sparring partners. Set Schedule & create Club Training Sessions, so others can attend.

Techniques Base

BJJ / Judo / Grappling Techniques open database. Every user can add up to 5 techniques before verification.

Profile

Create your profile and let others see you! Only people from your Club can see your profile
